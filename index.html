Rules and Regulations

1. All consequences for abuse of this system will be dealt with according to university policy.

2. Users must comply with all university policies and regulations.

3. Users must not use the system for any illegal or unauthorized purpose.

4. Users must not interfere with or disrupt the system or servers or networks connected to the system.

5. Users must not violate any applicable laws or regulations in connection with their use of the system.

6. Users must behave respectfully towards others while using the system.

7. If class schedules are not available, users must not assume that a room is available.

8. If a class is in session, users must not enter the room without permission.

9. Users must not damage or vandalize any property while using the system.

10. If users reserved a room, but a class session uses the room, users must vacate the room immediately.

11. Users must not share their login credentials with anyone. If user does share their login credentials, they are responsible for any actions taken under those credentials.

12. Leave the rooms in clean and tidy condition after use.
