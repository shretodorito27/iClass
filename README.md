# iClass

In iClass, our goal is to allow the students of SJSU the opportunity to find more accessible study spaces. The Student Union and MLK Library can get extremely crowded and inconvenient for a place to study, but it can be inapproproaite to enter classrooms as classes may occur at any time. 

This is where we come in, we at iClass have sourced publically available data on classroom schedules to present students and faculty at SJSU with the ability to find available classrooms and "reserve" them. This would increase productivity, reduce stress, and allow for coordination between students on where they book their rooms.

There is an additional layer of security in that only those with an official .sjsu domain email address may reserve rooms, so faculty will be able to view those who have booked rooms at certain times if they request such information.

Disclaimer: The publically sources information does not include every classroom or every timing. If the room is occupied, it is upto the user's discretion, and if there is a course of official SJSU task occurring, the user should leave. If any user abuses this system, they will be banned and if necessary reported to SJSU for further proceedings.
